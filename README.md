# 🏰 FanIsle (Love2D RPG Prototype)

A 2D pixel-art top-down RPG, sandbox, and survival game built using the Love2D framework and Lua. Inspired by *The Survivalists*, this game features resource extraction, tools crafting, survival indicators, and helpers automation (e.g. training little goblin spirits to gather materials).

---

## 🎮 Core Game Mechanics

1. **Survival Metrics:**
   * **Health:** Lost during combat or extreme hunger.
   * **Hunger:** Depletes over time; restored by eating berries or cooked meat.
   * **Stamina:** Expended by sprinting, swinging axes, or mining.

2. **Sandbox & Gathering:**
   * **Trees:** Chop with an Axe to extract Wood and Saplings.
   * **Rocks:** Mine with a Pickaxe to extract Stone and Iron Ore.
   * **Shrubs:** Gather for Grass Fiber and Berries.

3. **Crafting & Building:**
   * Craft items directly (e.g., wooden axes, grass ropes, simple torches).
   * Construct world structures: campfires for cooking, chests for storage, and wooden walls/doors for shelter.

4. **Goblin Helpers (Automation):**
   * Find wild goblins in the forest, feed them sweet berries to recruit them.
   * Teach them actions (Gather, Mine, Build, Fight) to automate resource extraction, similar to *The Survivalists'* monkey mechanics.

5. **RPG & Combat:**
   * Explore procedural/grid map biomes: Safe Grasslands, Dark Forests, and Orc Dungeons.
   * Fight aggressive creatures (Skeletons, Orc scouts, Bats) to harvest rare drops.
